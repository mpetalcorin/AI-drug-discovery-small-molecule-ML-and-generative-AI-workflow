# AI Drug Discovery Small Molecule ML and Generative AI Workflow

This repository contains a reproducible AI for Drug Discovery workflow for small-molecule design, integrating:

- simulated BindingDB-, ChEMBL-, and PDBbind-inspired datasets
- molecular descriptor analysis
- structure-aware protein-ligand feature modelling
- active versus inactive compound classification
- pIC50 regression
- explainable AI
- uncertainty and applicability-domain analysis
- multi-objective lead prioritisation
- active-learning triage
- generative AI molecule design
- RDKit-enabled molecular validation and visualisation
- manuscript-ready figures and supplementary tables

The project demonstrates an agentic and explainable AI4DD workflow that can be extended to real public or internal drug-discovery datasets after appropriate curation, licensing, assay harmonisation, and validation.
